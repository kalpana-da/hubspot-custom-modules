# Hero Banner Module

A custom Hero Banner module built for HubSpot CMS using HubL.

## Features

- Editable Heading
- Editable Description
- CTA Button
- Responsive Design
- Image Upload
- Dynamic Style Fields
- Optimized Images using `resize_image_url()`

## Technologies

- HubSpot CMS
- HubL
- HTML5
- CSS3

## Skills Demonstrated

- Custom Module Development
- Dynamic Fields
- Responsive Design
- HubL Templating
