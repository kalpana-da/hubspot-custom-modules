# HubSpot Custom Modules

A collection of custom HubSpot CMS modules built using HubL, HTML, CSS, and HubSpot CMS best practices. These modules were developed as part of hands-on learning and demonstrate reusable, responsive, and editor-friendly components.

## Modules Included

### 1. Hero Banner Module
- Responsive hero section
- Editable heading and description
- CTA button
- Image upload
- Dynamic style fields
- Image optimization using `resize_image_url()`

### 2. Team Members Module
- Repeater field for multiple team members
- Editable image, name, designation, and description
- Responsive card layout
- Dynamic styling

## Technologies Used

- HubSpot CMS
- HubL
- HTML5
- CSS3
- Responsive Web Design

## Skills Demonstrated

- Custom HubSpot Module Development
- HubL Templating
- `fields.json`
- `meta.json`
- `require_css`
- `scope_css`
- Repeater Fields
- Theme Development
- Responsive Design

## Preview

These modules were created as part of HubSpot CMS development practice and can be integrated into HubSpot themes and templates.
