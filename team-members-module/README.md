# Team Members Module

A reusable Team Members module built for HubSpot CMS using HubL. This module allows content editors to add multiple team members using a Repeater field, making it ideal for About Us and Team pages.

## Create the Module in HubSpot

1. Go to **Content → Design Manager**
2. Click **File → New File**
3. Select **Module**
4. Create a **Custom Module**
5. Replace the generated files with the files from this repository.

---

## Create Module Fields

| Field | Type |
|--------|------|
| Team Members | Repeater |
| Name | Text |
| Designation | Text |
| Description | Rich Text |
| Photo | Image |

---

## Add the Module to a Page

1. Drag the Team Members Module onto a page.
2. Add one or more team members.
3. Publish the page.
